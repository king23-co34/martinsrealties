4. **Save and upload to your server**

## 💡 IMAGE TIPS

- **Format:** JPG works great for photos (smaller file size)
- **Quality:** Use high-quality images (but optimize for web - max 500KB per image)
- **Mobile:** Make sure images look good on phone screens too
- **Aspect Ratios:** Follow the recommended sizes for best appearance
- **Alt Text:** Already included in HTML for SEO

---

## ⚠️ IMPORTANT NOTES

- Upload images with **EXACT filenames** (case-sensitive on some servers)
- Use **JPG format** for best performance
- Images should be **optimized for web** (compressed but high quality)
- If an image fails to load, it will show a **beautiful colored gradient** fallback
